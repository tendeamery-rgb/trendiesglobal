# Trendies owner editing guide

This site includes an owner editor at:

`https://trendiesglobal.com/admin/`

Use it to update homepage words, swap photos and add blog posts without touching code.

This works on Netlify Free. A drag-and-drop deploy can launch the site today; the no-code editor needs the site connected to GitHub so it has somewhere to save your future edits.

## One-time Netlify setup

The editor can only save changes if the site is connected to a GitHub repo in Netlify. A drag-and-drop deploy will still show the website, but the editor will not be able to save.

1. In Netlify, connect this folder/repo to GitHub.
2. In Netlify, go to Identity and enable Identity.
3. Go to Identity > Services and enable Git Gateway.
4. Invite your own email as a user.
5. Open `https://trendiesglobal.com/admin/` and log in.

## What you can edit

- Homepage headline and intro text
- Main homepage photos
- The live hub cards
- Vision, join, flag wall, wall, story and partner copy
- Wall prompt buttons
- Blog posts / chapter notes

## Photo tips

- Use JPG or PNG.
- Keep homepage photos under about 1 MB when possible.
- Add clear alt text, for example: `Friends having a picnic in London`.
- For the three interest-form side photos, use clear photos where the important person is near the centre. The site now protects these photos from hard cropping, but centred subjects will always look best.
- After saving, wait for Netlify to finish deploying before refreshing the live site.

## Blog tips

Use blog posts for short updates: launch notes, community ideas, safe plan guidelines, city reflections and behind-the-scenes notes from Chapter One.
